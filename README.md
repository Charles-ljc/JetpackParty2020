# JetpackParty
Jetpack聚会

此项目依赖了所有Jetpack库，可以方便地搜到Jetpack库中的任何一个类，从而找到此类的依赖配置

https://developer.android.google.cn/jetpack
https://developer.android.google.cn/jetpack/androidx/versions
https://maven.google.com/web/index.html

使用assembleDebug命令运行
